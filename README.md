Ответы на вопросы:
o Какие jQuery-функции использовались?
$('html, body').animate()-Плавная прокрутка 
$().draggable() - для перетаскивания элемента 
$().datepicker() - Превращает input в календарь (jQuery UI)
$().offset()- Получение позиции элемента для прокрутки

### Что проще — jQuery или чистый JS?
jQuery проще и короче для быстрых скриптов.

### Что бы использовал сегодня?
Для современных сайтов — Vanilla JS и Fetch API.  
jQuery — только для legacy-проектов.
